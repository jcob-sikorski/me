# Aria Template

This is a personal blog, portfolio, and blog template created for [Astro](https://astro.build).

This project is an Astro port of the original [aria template](https://github.com/static-templates/aria).

## Quick Start

1. Clone this repository
2. Install dependencies:
```bash
   npm install
```
3. Run the development server:
```bash
   npm run dev
```

- Responsive design
- Blog functionality
- Portfolio showcase
- Easy to customize
